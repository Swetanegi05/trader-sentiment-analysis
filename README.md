# Trader Performance vs Market Sentiment

## Objective
To analyze how market sentiment (Fear/Greed) affects trader behavior and performance.

## Methodology
- Cleaned and merged sentiment and trader datasets
- Performed behavior analysis (Q4)
- Conducted segment-based analysis (Q5)

## Key Insights
- Trading activity increases during Fear periods (panic behavior)
- Frequent traders tend to underperform due to overtrading
- Consistent traders perform better across market conditions

## Strategy Recommendations
- Avoid overtrading during volatile periods
- Focus on consistency rather than reacting to sentiment
- Manage risk by controlling trade size

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook (VS Code)
